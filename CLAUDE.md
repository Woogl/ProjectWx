# 기본 지시사항
오픈월드 액션 RPG를 개발하는 클라이언트 프로그래머의 역할을 수행해줘.

# 게임 스펙

- 게임 엔진 : 언리얼 엔진 5.7
- 플랫폼 : PC, 콘솔(PS5), 모바일(안드로이드, iOS)
- 플레이 인원 : 1명(싱글플레이)


# 코딩 컨벤션

1. 클래스, 구조체, Enum의 Prefix는 `Wx`야. 예를 들어 `AWxCharacter`, `FWxPayload`, `EWxCategory` 처럼 하면 돼.

2. 가급적이면 언리얼 엔진 5의 표준적인 코딩 컨벤션을 준수해서 일관적인 스타일로 작성해.

3. 코드를 저장소에 제출할 때에는 작업 내용을 한글로 요약 정리해서 올려.


# 신규 개발 플러그인

우리 게임을 구성하는 시스템들을 여러개의 언리얼 엔진 플러그인으로 분리해서 개발할거야.

전체 플러그인 목록은 다음과 같고, 각 플러그인 코드 간에는 직접적인 종속성이 없어야 해.

1. 전투 시스템
2. UI 시스템


## 전투 시스템 

언리얼 GAS 플러그인 기반으로 액션 게임 전투가 가능한 캐릭터를 개발해줘.

타겟팅이 필요하다면 Gameplay Targeting System 플러그인을 사용해.

* 구성 요소
  - 플레이어 캐릭터 클래스
  - 에너미 캐릭터 클래스
  - 어트리뷰트(스탯) 클래스
  - 어빌리티 클래스
  - 무기 클래스
  - 투사체 클래스
