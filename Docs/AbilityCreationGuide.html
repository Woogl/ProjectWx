핵심 구성 요소

Blueprint WxAbility

어빌리티의 로직을 정의합니다. 언제 발동하고, 무엇을 하는지 결정합니다.

애니메이션 AnimMontage

어빌리티가 재생하는 애니메이션입니다. Notify를 배치할 수 있습니다.

구간 AnimNotifyState

몽타주의 특정 구간에 배치하여 상태를 활성화합니다. (무기 충돌, 콤보 입력 등)

시점 AnimNotify

몽타주의 특정 시점에 이벤트를 발생시킵니다. (투사체 발사 등)

수치 GameplayEffect

데미지, 버프, 디버프 등의 수치 변경을 담당합니다.

데이터 AbilitySet

캐릭터에 부여할 어빌리티/이펙트 목록을 정의하는 데이터 에셋입니다.

작업 흐름

1 AnimMontage 준비 애니메이션 에셋
➔

➔

➔

TurnAround

캐릭터를 타겟 방향으로 부드럽게 회전시킵니다.
주로 공격/스킬 시작 구간에 배치합니다.

TargetingPreset타겟 탐색에 사용할 타겟팅 프리셋

WeaponCollision

무기의 데미지 판정을 활성화합니다.
이 구간 동안만 무기가 적에게 데미지를 줍니다.

💡 구간이 너무 넓으면 부자연스러운 판정이 발생합니다. 실제 스윙 프레임에 정확히 맞추세요.

ComboWindow

콤보 입력을 받는 구간입니다.
이 구간에서 공격 입력 시 다음 콤보로 이어집니다.

💡 현재 WxAbility_Attack에서만 사용 가능합니다.

Invincible

무적 상태를 부여합니다. 회피의 핵심 이동 구간이나 스킬 선딜에 사용합니다.

Guard

가드 상태를 활성화합니다. 피격 시 가드 전용 HitReact가 발동됩니다.

SpawnProjectile AnimNotify

특정 시점에 투사체를 생성합니다. 구간이 아닌 단일 시점에 배치합니다.
락온 중이면 투사체가 자동으로 락온 대상 방향으로 발사됩니다.

SpawnSocketName투사체가 생성될 소켓 (예: hand_r)

자주 묻는 질문

DamageEffectClass는 어디서 설정하나요?

무기 BP(WxWeaponBase 기반)와 투사체 BP(WxProjectileBase 기반) 모두 DamageEffectClass 속성을 가지고 있습니다.
여기에 적절한 GameplayEffect를 설정해야 데미지가 적용됩니다.

새로운 Gameplay Tag가 필요하면 어떻게 하나요?

Gameplay Tag는 C++ 코드에서 선언해야 합니다. 필요한 태그를 프로그래머에게 요청하세요.

콤보 공격의 속도를 조절하려면?

AnimMontage의 PlayRate를 조절하거나, 몽타주 에디터에서 애니메이션 속도를 변경합니다.
Notify 구간도 함께 조정해야 합니다.

하나의 캐릭터에 여러 공격 어빌리티를 넣을 수 있나요?

가능하지만, 같은 ActivationInputTag를 사용하면 충돌합니다.
여러 공격 어빌리티를 넣으려면 서로 다른 입력 태그를 사용해야 합니다.

적 캐릭터의 어빌리티는 어떻게 발동시키나요?

적 캐릭터는 Behavior Tree의 BT Task에서 ASC를 통해 어빌리티를 발동합니다.
